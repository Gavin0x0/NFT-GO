# NFT-GO
## 基于Vue3.0+Eelement-Plus+FastAPI+MongoDB的前后端分离的简易商城项目「前端部分」
 -  项目重点在于试水目前主流的前端框架，快速开发简易的在线服务
## 运行环境
 - node.js
 - vue 3.0
## 启动前安装依赖
```
yarn install
```

### 本地启动
```
yarn serve
```

### 打包
```
yarn build
```

### 代码检查
```
yarn lint
```
